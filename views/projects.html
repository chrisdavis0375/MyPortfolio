Train Scheduler

Train Scheduler allows you to be able to manage different trains estimated time of arrival, the frequency at which they travel, and their destinations.

How it works:
The user can add a new train to the scheduler by navigating to the bottom of the page and typing in a new trains name, destination, frequency, and ETA. After clicking submit the new train will appear in the train table and also be stored in a firebase database.

Technologies used:
HTML5, CSS, Bootstrap, Javascript, jQuery, and Firebase

Product Genie

Product Genie is a searching app that allows the user to search for a wide variety of different items using Ali Seeks API. Genie also returns a list of articles related to the users search mixed with reviews.

How it Works:
Once the user searches for a product using the search box Ali Seeks API will retrieve a list of results and return them to the user on the first row of the page along with recent searches followed by articles and reviews that have working links that can redirect the user to another tab.

Technologies used:
HTML5, CSS, Animista, Bootstrap, Javascript, jQuery, and Firebase

Star Wars RPG

This RPG allows the user to play as one of four iconic star wars characters and duel with the remaining three. In order to win the user must choose their opponents order wisely base on their stats.

How it works:
When the user selects the character they would like to use, the remaining three characters will become opponents. The user will then have to select an opponent which will then become a defender. When the user clicks the attack button damage will be inflicted on the opponent while the opponent inflicts counter damage. Once the opponents health is at 0 the user will be able to select another opponent. If all opponents are defeated the user wins.

Technologies used:
HTML5, CSS, and Javascript
